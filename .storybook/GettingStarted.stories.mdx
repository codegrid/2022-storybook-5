# はじめに

CodeGridの[Storybook入門](https://www.codegrid.net/series/2022-storybook)のデモになります。

## インストールとStorybookの起動

対象のリポジトリをクローンしてパッケージをインストールしたら、Storybookを起動してください。

```
npm run storybook
```

## コンポーネント

```jsx
  ボタン
</Button>
```
